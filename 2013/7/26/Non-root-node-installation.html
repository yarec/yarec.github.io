非root安装node


通常的linux环境是不会直接提供root用户，这种情况下想要安装node其实也很简单。只要在configure里指定PREFIX就可以了。
下面是完整的安装命令：

git clone https://github.com/joyent/node.git -b master
cd node
mkdir ~/opt
export PREFIX=~/opt; ./configure
make
make install
echo 'export PATH=~/opt/bin:${PATH}' >> ~/.bashrc
source ~/.bashrc

 测试一下：

$ node -v
v0.11.5-pre
$ npm -v
1.3.4

再测试一下webapp，以下代码保存到app.js

var http = require('http');
server = http.createServer(function (req, res) {
res.writeHeader(200, {"Content-Type": "text/plain"});
res.end("Hello Node");
})
server.listen(3000);

执行：node app.js
在浏览器中访问 http://127.0.0.1:3000
显示“Hello Node”说明一切正常。

试用Express:

npm install -g express
mkdir site && cd site
express .
cd . && npm install
node app.js

在浏览器中访问 http://127.0.0.1:3000
