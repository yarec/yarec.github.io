快速安装centos


启动安装内核：

window用户推荐使用https://code.google.com/p/burg4dos/来设置，/isolinux目录可以从安装镜像中找到，复制到要安装的机器。

grub配置：

1. grub-legacy

title install linux
root (hd0,0)
kernel /isolinux/vmlinuz
initrd /isolinux/initrd.img

2. grub2

set root=(hd0,0)
linux /isolinux/vmlinuz
initrd /isolinux/initrd.img
boot

安装选项：

启动之后，经过一些基本的选择就到达“Installation Methed”界面。这个地方我喜欢使用URL方式，即是在局域网内部署一个http服务，把centos镜像解压后放在一个目录中。这种方法适合经常安装，不用把好几个G的iso拷来拷去的。设置好url后，自动下载一个“/images/install.img”文件，这个是安装主程序。安装程序运行以后，其它的就方便操作了。

配置网络：

以网络安装的方式是不需要这一步的，这里对应的是使用虚拟机安装时，安装完成后，用下面命令来配置网络

ifconfig eth0 192.168.1.101 up
