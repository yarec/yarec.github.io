sublime text配置


安装Package Control

sublime_package_control：　A full-featured package manager for Sublime Text　https://sublime.wbond.net

Ctrl+`调出python控制台，输入下面代码并回车。

import urllib.request,os; pf = 'Package Control.sublime-package'; ipp = sublime.installed_packages_path(); urllib.request.install_opener( urllib.request.build_opener( urllib.request.ProxyHandler()) ); open(os.path.join(ipp, pf), 'wb').write(urllib.request.urlopen( 'http://sublime.wbond.net/' + pf.replace(' ','%20')).read())

安装插件：

ctrl+shift+p 打开Package Control组件，输入install 调出 Install Package 选项并回车，然后输入要安装的插件名称（如：jsformat）即可。

常用插件：

js formater

Zen Coding

SublimeLinter

Alignment

Bracket Highlighter

windows常用命令


查看进程信息：tasklist /V
查看占用８０端口的进程号：Netstat -ano|findstr "80"
查看进程号为４２５２的进程信息：tasklist|findstr "4252"

建立一个用户：net user uname upass /add
删除一个用户：net user uname /del
把用户添加到管理员：net localgroup administrators uname /add
把用户从管理员组删除：net localgroup administrators uname /del
查看所有用户账户：net user
查看当前用户：whoami | echo %username%
查看远程用户：query user（需要管理员身份）

设置订单任务：schtasks /create /sc minute /mo 1 /tn  "time test" /tr c:\test.bat

编辑文件：copy con test.bat (CTRL+Z 保存退出)

linux查看硬件信息


linux查看硬件信息的命令

软件类：

用硬件检测程序kuduz探测新硬件：service kudzu start ( or restart)
查看系统硬盘信息和使用情况：fdisk & disk - l & df & hdparm -I /dev/sda 
dmidecode查看硬件信息，包括bios、cpu、内存等信息
dmesg | more 查看硬件信息
lshw(Hardware Lister可以把硬件的详细信息导出

proc信息类：

查看CPU信息：cat /proc/cpuinfo
查看板卡信息：

  1. cat /proc/pci
  2. lspci (相比cat /proc/pci更直观） 例子： lspci |grep Ethernet 查看网卡型号

查看内存信息：cat /proc/meminfo
查看USB设备：

  1. cat /proc/bus/usb/devices
  2. lsusb

查看键盘和鼠标:cat /proc/bus/input/devices
查看各设备的中断请求(IRQ):cat /proc/interrupts

OS信息类：

查看系统体系结构：uname -a
查看当前操作系统发行版信息：cat /etc/issue
查看系统位数（32/64）：getconf LONG_BIT
查看系统负载：cat /proc/loadavg

ubuntu拨号上网和wifi共享


ubuntu的网络管理器和windows没多大差异，挺方便的。但是我遇到两个问题（可能只是少数人的需求）：

  1. 拨号上网
  2. 共享wifi

解决方法
一、拨号上网的方法： pppoeconf

  1. 打开终端，Ctrl+Alt+t
  2. 以root身份启动，执行 sudo pppoeconf
  3. 启动后会提示一些问题，一路选“Yes"即可，直到要求输入ADSL账号，填好账号，下一步是密码，再之后一路"Yes"就ok了
  4. 其中最后一步是设置是否开机自动拨号，如果不想自动拨就选“No”，手动拨号的命令是 pon dsl-provider

我一直是用的这个方法，另外网上有人说有图形操作的方法： 系统-》首选项-》网络连接 选择 DSL 然后添加 输入用户名和密码 勾上自动连接 在点应用 就OK 了
这个方法我试了不好使，不知道是不是得重启电脑才行，前一种方法已经满足我了，不折腾了。
pppoeconf和NetworkManager是相冲突的，用过pppoeconf的想要换为nm需要下面的设置（此方法我没有试成功，大家可以试试）：

  1. 更改/etc/NetworkManager/NetworkManager.conf中的managed＝true
  2. 更改/etc/network/interfaces，只保留
    auto lo
    iface lo inet loopback
  3. 删除，rm /etc/resolv.conf
  4. 重启nm，sudo /etc/init.d/network-manager restart （如果面板没有nm图标，ALT+F2然后输入nm-applet回车）

一些命令：

  • Ubuntu ADSL手工断线：sudo poff
  • 查看连接情况：sudo plog
  • 可查询本机的上网IP: sudo ifconfig ppp0

二、windows上共享wifi有手动的方法，也有好用的工具。ubuntu上手动的方法，一直没有成功。最后发现一个工具， wifi-hostapd-ap，非常方便的工具啊。我这里是编译安装的。先安装依赖的库： sudo apt-get install wireless-tools bridge-utils hostapd dnsmasq libqtgui4
编译：make
启动：./WiFi_Hostapd_AP
设置密码=》应用， 搞定！
